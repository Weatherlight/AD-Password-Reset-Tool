# AD Password Reset Tool (GUI)

Active Directory のユーザーパスワードを CSV ファイルから一括でリセットするための PowerShell ツールです。
GUI を備えているため、エンジニア以外でも直感的に操作が可能です。

## 🚀 主な機能

- **GUI 操作**: ファイル選択や実行をボタン一つで行えます。
- **CSV 一括処理**: 複数ユーザーのパスワードを一度に更新します。
- **ログ出力**: 実行結果（成功・失敗）をリアルタイム表示し、ファイルとして保存します。
- **ログ出力先の変更**: 任意のフォルダをログ保存先に指定可能です。
- **安全設計**: ユーザーの存在確認を行い、エラー時には詳細な理由を記録します。

## 📋 動作要件

- **OS**: Windows 10 / 11, Windows Server 2016 以降
- **PowerShell**: 5.1 以上
- **必須モジュール**: `ActiveDirectory` モジュール (RSAT)
- **権限**: 対象ユーザーのパスワードリセット権限を持つアカウントでの実行
